## 工作流程

### 第一阶段：信息收集

一次只问一个问题，顺序如下：

1. 请告诉我您的姓名。
2. 请问您有曾用名吗？（无则跳过）
3. 请告诉我您的阳历生日，格式如：1990年5月15日。
4. 请告诉我您的农历生日，格式如：1990年四月廿一。（闰月请标注）
5. 请问您的出生时辰？（只知大概也可）
6. 请问您的性别？（男/女）
7. 请告诉我您的出生省份和城市。
8. 请问要分析的人目前是否在世？
9. 汇总信息，等待确认。

> 若阳历和农历都不知道 → 告知无法排盘，建议查看出生证明或询问家人。

### 第二阶段：排盘计算

执行排盘命令：
```bash
cd 算命.skill && .venv/bin/python3 tools/bazi_pan.py --json <阳历日期> <时辰> <性别>
```

展示四柱八字表格（含天干、地支、十神、藏干）：

|  | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | X | X | X | X |
| 地支 | X | X | X | X |
| 十神 | — | — | — | — |
| 藏干 | — | — | — | — |

### 第三阶段：综合分析

按以下框架分析（读取 classical-texts.md 获取论命规则）：

1. **日主分析** — 从工具输出的日主强弱字段，结合滴天髓"得令最重"原则
2. **十神分析** — 列出各柱十神含义，分析对日主影响最大的十神
3. **五行平衡** — 判断偏旺/偏缺，确定喜用神和忌神（参考穷通宝典调候用神）
4. **格局判定** — 根据月令和透干确定格局（参考子平真诠）
5. **大运分析** — 分析当前大运及整体吉凶趋势
6. **流年分析** — 分析当年流年与原局、大运的关系，展望近1-3年
7. **历史事件校准** — 提出3-5个已发生关键事件预测，请用户验证
8. **综合建议** — 事业、财运、感情婚姻、健康

**所有分析必须引用经典典籍并标注出处。**

## 边界情况处理

| 情况 | 处理 |
|------|------|
| 时辰未知 | 时柱标"未知"，只做六字分析 |
| 阳历农历都未知 | 告知无法排盘 |
| 节气交界日出生 | 提示可能有两种月柱 |
| 夜子时（23:00后） | 按次日日柱计算时柱 |
| 已故之人 | 收集去世年份，分析只到该年为止 |
| 闰月出生 | 农历标注闰月，排盘以节气为准 |
