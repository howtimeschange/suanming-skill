## Python 排盘工具

**工具路径**：`tools/bazi_pan.py`
**Python 环境**：`.venv/bin/python3`（在 skill 目录下执行）

### 基本命令

```bash
cd ~/.openclaw/workspace/算命.skill
.venv/bin/python3 tools/bazi_pan.py --json <阳历日期> <时辰> <性别>
# 例：.venv/bin/python3 tools/bazi_pan.py --json 1990-5-15 8:00 male
```

### JSON 输出字段

| 字段 | 说明 |
|------|------|
| `四柱` | 年柱/月柱/日柱/时柱（干支） |
| `十神` | 各柱十神（以日干为基准） |
| `藏干` | 各支本气/中气/余气 |
| `五行` | 金木水火土数量统计 |
| `纳音` | 各柱纳音五行 |
| `大运` | 方向、起运年龄、完整10步大运列表 |
| `流年` | 近5年流年干支 |
| `日主强弱` | 得令/得地判断、强弱判定、调候倾向 |

## 第一阶段：信息收集

一次只问一个问题。

1. **姓名**：请告诉我您的姓名。
2. **曾用名**：请问您有曾用名吗？（没有则跳过）
3. **阳历生日**：请告诉我您的阳历（公历）生日，格式如：1990年5月15日。
4. **农历生日**：请告诉我您的农历生日，格式如：1990年四月廿一。（闰月请标注）
5. **出生时辰**：请问您的出生时辰？（只知大概也可，如"早上"）
6. **性别**：请问您的性别？（男/女）
7. **出生地**：请告诉我您的出生省份和城市。
8. **在世状态**：请问要分析的人目前是否在世？
9. **信息确认**：汇总展示，等待用户确认后再排盘。

> ⚠️ 若阳历和农历都不知道 → 告知无法排盘，建议查看出生证明或询问家人。

## 第三阶段：综合分析

读取 `references/classical-texts.md`，按以下框架分析：

### 1. 日主分析
从工具输出的 `日主强弱` 字段获取，结合滴天髓"得令最重"原则。

### 2. 十神分析
列出各柱十神含义，重点分析对日主影响最大的十神。

### 3. 五行平衡
判断五行偏旺/偏缺，确定喜用神和忌神（参考穷通宝典调候用神）。

### 4. 格局判定
根据月令和透干确定格局（正官格、偏财格等，参考子平真诠）。

### 5. 大运分析
分析当前所处大运及整体吉凶趋势。

### 6. 流年分析
分析当年流年干支与原局、大运的关系，展望近1-3年。

### 7. 历史事件校准（关键步骤）
提出 3-5 个已发生的关键事件预测，请用户验证。根据反馈微调分析。

### 8. 综合建议
事业方向、财运趋势、感情婚姻、健康注意事项。

**所有分析必须引用经典典籍并标注出处。**

## 边界情况

| 情况 | 处理 |
|------|------|
| 时辰未知 | 时柱标"未知"，只做六字分析 |
| 阳历农历都未知 | 告知无法排盘 |
| 节气交界日出生 | 提示可能有两种月柱 |
| 夜子时（23:00后） | 按次日日柱计算时柱 |
| 已故之人 | 收集去世年份，分析只到该年为止 |
| 闰月出生 | 农历标注闰月，排盘以节气为准 |
