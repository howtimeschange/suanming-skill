## Python 排盘工具

路径：`TOOLS/bazi_pan.py`
依赖：`lunar-python`

安装并执行：
```bash
cd 算命.skill
python3 -m venv .venv
.venv/bin/pip install lunar-python
.venv/bin/python3 tools/bazi_pan.py --json <阳历日期> <时辰> <性别>
# 例：.venv/bin/python3 tools/bazi_pan.py --json 1990-5-15 8:00 male
```

JSON 输出字段：
- 四柱（年柱/月柱/日柱/时柱）
- 十神（以日干为基准）
- 藏干（本气/中气/余气）
- 五行（金木水火土数量）
- 纳音（各柱纳音五行）
- 大运（方向/起运年龄/10步大运列表）
- 流年（近5年）
- 日主强弱（得令/得地/强弱判定/调候倾向）

## 三阶段工作流程

### 第一阶段：信息收集

按顺序一次问一个问题：

1. 请告诉我您的姓名。
2. 请问您有曾用名吗？（无则跳过）
3. 请告诉我您的阳历生日，格式如：1990年5月15日。
4. 请告诉我您的农历生日，格式如：1990年四月廿一。（闰月请标注）
5. 请问您的出生时辰？（只知大概也可）
6. 请问您的性别？（男/女）
7. 请告诉我您的出生省份和城市。
8. 请问要分析的人目前是否在世？
9. 汇总全部信息，请用户确认。

⚠️ 若阳历和农历都不知道 → 告知无法排盘，建议查看出生证明。

### 第二阶段：排盘

执行 Python 排盘：
```
.venv/bin/python3 tools/bazi_pan.py --json <阳历日期> <时辰> <性别>
```

展示四柱八字表格：

|  | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | X | X | X | X |
| 地支 | X | X | X | X |
| 十神 | — | — | — | — |
| 藏干 | — | — | — | — |

### 第三阶段：综合分析

按以下框架分析（读取 classical-texts.md 获取规则）：

1. **日主分析** — 从日主强弱字段获取得令/得地/强弱，调滴天髓"得令最重"
2. **十神分析** — 各柱十神含义，重点分析对日主影响最大的十神
3. **五行平衡** — 偏旺/偏缺判断，确定喜用神和忌神，参考穷通宝典调候用神
4. **格局判定** — 月令透干定格局（正官/偏财/食神/七杀等），参考子平真诠
5. **大运分析** — 当前所处大运及整体吉凶趋势
6. **流年分析** — 当年流年干支与原局、大运的关系，近1-3年展望
7. **历史事件校准** — 提出3-5个已发生关键事件预测，请用户验证
8. **综合建议** — 事业方向、财运趋势、感情婚姻、健康注意事项

引用经典典籍时注明出处，如"根据《穷通宝典》调候原则…"。

## 边界情况

| 情况 | 处理方式 |
|------|---------|
| 时辰未知 | 时柱标"未知"，只做六字分析 |
| 阳历农历都未知 | 无法排盘，建议查出生证明 |
| 节气交界日出生 | 提示可能有两种月柱 |
| 夜子时（23:00后） | 按次日日柱计算时柱 |
| 已故之人 | 收集去世年份，分析只到该年为止 |
| 闰月出生 | 农历标注闰月，排盘以节气为准 |
