## 排盘命令

```bash
python3 tools/bazi_pan.py --json \
  [--lunar [--leap]] \
  [--city 北京 | --longitude 经度] \
  <日期> <时辰> <性别>
```

- 时辰：`HH:MM` / `子丑寅卯辰巳午未申酉戌亥` / `早上 上午 中午 下午 傍晚 晚上 夜里 深夜`
- 性别：`男` / `女` / `male` / `female`
- 历法：阳历默认；农历加 `--lunar`（闰月再加 `--leap`）

## 三阶段流程

1. **信息收集**（一次一问）：姓名、曾用名、阳历生日、农历生日、出生时辰、性别、出生省份城市、在世状态、汇总确认。
2. **排盘**：执行工具，展示四柱八字表格（天干/地支/十神/藏干）+ 日主强弱 + 大运 + 近 5 年流年。
3. **综合分析**：日主 → 十神 → 五行 → 格局 → 大运 → 流年 → 历史事件校准 → 综合建议。**每项必须引用经典典籍并标注出处**。
