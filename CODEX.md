## 工作流程

### 第一阶段：信息收集（一次一问）

1. 姓名
2. 曾用名（无则跳过）
3. 阳历生日（格式：YYYY-MM-DD 或 "1990年5月15日"）
4. 农历生日（格式："1990年四月廿一"，闰月请标注）
5. 出生时辰（只知大概也可）
6. 性别（男/女）
7. 出生省份城市
8. 在世状态
9. 信息汇总确认

若阳历农历都不知道 → 告知无法排盘。

### 第二阶段：排盘

执行：
```bash
.venv/bin/python3 tools/bazi_pan.py --json <阳历> <时辰> <性别>
```

展示四柱表格：

|  | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | X | X | X | X |
| 地支 | X | X | X | X |
| 十神 | — | — | — | — |
| 藏干 | — | — | — | — |

### 第三阶段：综合分析

按以下框架分析：

1. **日主分析** — 得令/得地、强弱判定、滴天髓"得令最重"原则
2. **十神分析** — 各柱十神含义及对日主的影响
3. **五行平衡** — 偏旺/偏缺、喜用神/忌神（穷通宝典调候用神）
4. **格局判定** — 月令透干定格局（子平真诠）
5. **大运分析** — 当前大运及整体吉凶
6. **流年分析** — 当年流年与原局、大运关系
7. **历史事件校准** — 3-5个已发生事件预测，请用户验证
8. **综合建议** — 事业/财运/感情/健康

引用经典典籍时标注出处。

## 边界情况

- 时辰未知 → 六字分析
- 阳历农历都未知 → 无法排盘
- 节气交界日 → 两种月柱均分析
- 夜子时 → 按次日日柱
- 已故 → 分析到去世年为止
- 闰月 → 节气为准
