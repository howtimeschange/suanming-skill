# 常用神煞速查

神煞不决定吉凶，只是辅助性标记，分析时作为"意象标签"参考，切忌以神煞论命。

## 贵人类

### 天乙贵人（最尊贵）
以日干查年支/月支/日支/时支：

| 日干 | 天乙贵人地支 |
|------|-------------|
| 甲、戊、庚 | 丑、未 |
| 乙、己 | 子、申 |
| 丙、丁 | 亥、酉 |
| 壬、癸 | 巳、卯 |
| 辛 | 午、寅 |

意象：逢凶化吉、贵人相助、官场得力。

### 文昌贵人
以日干查地支：

| 日干 | 文昌 |
|------|------|
| 甲 | 巳 |
| 乙 | 午 |

| 丁、己 | 酉 |
| 庚 | 亥 |

## 桃花 / 咸池

以日支或年支三合局的"子午卯酉"位：

| 三合局 | 桃花 |
|--------|------|
| 申子辰 | 酉 |
| 亥卯未 | 子 |
| 寅午戌 | 卯 |

意象：异性缘、魅力、艺术天赋；在子女/配偶宫主风流。

## 空亡（旬空）

按日柱所在旬查：某旬天干用完后剩下的两支即空亡。

| 日柱所在旬 | 空亡地支 |
|-----------|---------|
| 甲子旬（甲子~癸酉） | 戌、亥 |
| 甲戌旬（甲戌~癸未） | 申、酉 |
| 甲申旬（甲申~癸巳） | 午、未 |
| 甲午旬（甲午~癸卯） | 辰、巳 |
| 甲辰旬（甲辰~癸丑） | 寅、卯 |
| 甲寅旬（甲寅~癸亥） | 子、丑 |

意象：该宫位能量空虚、事物易成空；忌神空亡反为吉。

---

## 使用提醒

- 神煞百种繁多，不宜见煞必断凶、见贵必断富。
- 以滴天髓"喜用为贵，忌神为煞"为原则，结合神煞的意象丰富分析色彩即可。
- 神煞判断可由 LLM 在分析阶段按需计算，工具不强制输出。
