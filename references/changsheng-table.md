# 十二长生表

十二长生描述某天干在不同地支环境下的能量盛衰曲线，用于判断日主旺衰、通根、调候。

## 阶段顺序

长生 → 沐浴 → 冠带 → 临官 → 帝旺 → 衰 → 病 → 死 → 墓 → 绝 → 胎 → 养

- **长生**：新生初现，根气初萌
- **沐浴**：稚嫩易损，桃花气重
- **冠带**：渐入佳境
- **临官（禄）**：当权得位
- **帝旺**：能量巅峰
- **衰**：转势下降
- **病 / 死**：能量衰竭
- **墓**：收藏入库
- **绝 / 胎 / 养**：休息孕育
